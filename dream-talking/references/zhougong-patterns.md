# 周公/民间解梦案例提炼

使用方式：这是传统象征层的校准资料，不是事实预测。输出时只能说“传统上常被理解为”“民间解梦里常和……有关”“从传统象征看，可能偏向……”。不要把下面内容写成确定预兆，也不要建议用户据此做现实决策。

资料来源以公开网页和古籍整理页为参考，已经转写为可复用的判断模式，不直接搬运原文。

## 总体规律

民间解梦常按“意象 + 动作 + 结局”判断，而不是只看名词：

- 同一意象在不同动作里可能完全相反，比如“水清”和“被水淹没”，“蛇路过”和“蛇追咬”。
- 传统层常把自然物解释成财、运、变化、家宅、人情；现代层要把它转成情绪、压力、关系和身份感。
- 古籍词条里常出现吉凶断语。输出时不要照搬“主凶”“大吉”等强断言，要翻译成温和倾向。

推荐转换：

- “大吉/得财/升迁” → “传统上偏向机会、资源、进展感”。
- “凶/疾病/争讼” → “传统上偏向压力、阻碍、安全感议题”。
- “家宅不安/分家” → “可能和家庭边界、关系变动、归属感有关”。
- “贵人/尊重/人缘” → “可能和支持、被看见、关系资源有关”。

## 蛇

传统高频含义：变化、财气、贵人、欲望、警觉、家宅不安、口舌压力。

动作区分：

- 蛇只是出现、路过、远离：传统上可偏向变化靠近、资源流动、警觉被唤起。
- 被蛇咬：传统词条常有“交好运/生子”等说法；现代输出更适合转成“某件事触碰到你，让你有压力也有转机”。
- 打死蛇、杀蛇：传统上多偏向排除困难、压住麻烦、克服阻碍。
- 很多蛇、群蛇：传统上有吉利说法，也有宅不安、口舌压力说法；结合情绪判断，害怕时偏压力堆叠，不怕时偏变化密集。
- 蛇进屋、在床、在家里：传统上常连到家宅、亲密关系、身体安全感；现代层要问“家里/伴侣/隐私边界是否有压力”。
- 蛇在水里、捉蛇：传统上常连到财利、功名、流动中的机会；现代层可解释为“在情绪变化里试图抓住机会”。

输出提醒：蛇类梦很容易被讲成恐吓或性暗示。除非用户梦里有明确亲密/身体情绪，不要主动往性解释上靠。

## 水

传统高频含义：财、情绪流动、变化、滋养、阻碍、风险。

状态区分：

- 清水、泉水、平静水面：传统上偏吉，现代层偏情绪恢复、状态变清、关系顺。
- 大水、洪水、被淹没：传统上常有压力、冲击、灾象说法；现代层偏情绪过载、现实压力变大、控制感不足。
- 掉进水里但出来/没事：传统层可说“有惊无险”；现代层偏压力中仍有自救感。
- 喝水、洗澡、在水中自在：传统层偏滋养、恢复；现代层偏放松、补给、需要清理情绪。
- 脏水、臭水、浑水：传统层偏阻滞和不适；现代层偏消耗、烦心事、关系或环境不清爽。

输出提醒：不要把水直接等同发财。优先按水的清浊、深浅、是否失控来判断。

## 家、老家、房屋

传统高频含义：家宅、根基、财产、安全感、基本需要、人际边界。

场景区分：

- 回老家：传统和现代站点都常连到思念、根、安全、基本需要；现代层偏“回到原点/旧身份/熟悉关系”。
- 房屋稳固、回到熟悉的家：偏安全感、归属感、状态回收。
- 房屋损坏、漏水、倒塌、找不到家：偏安全感受损、现实根基不稳、家庭或身份焦虑。
- 搬家、迁居：传统层常连到变化和家人关系；现代层偏生活阶段转换、边界重组、想改变处境。
- 梦见失去房子/车子/工作：传统上可连到根基、财产、安全；现代层重点是“代价感”和“稳定感焦虑”。

输出提醒：家宅类梦尤其适合追问现实近况，如家庭责任、搬家、工作稳定、婚姻育儿压力。

## 亲戚、家人

传统高频含义：人缘、人情、贵人、家族关系、长辈支持、家庭压力。

关系区分：

- 多年没见的亲戚：传统层可说旧人旧事、人情关系浮现；现代层偏旧身份、家族记忆、归属感。
- 和亲戚亲近：偏家庭关系修复、支持、被接纳。
- 和亲戚争吵、尴尬：偏家庭角色压力、沟通不顺、边界感。
- 长辈教导、给东西：偏支持、经验、被提醒；现代层可转成“内在长辈声音”。
- 去世亲人：传统词条常有托梦、暗示、吉凶说法；本 skill 一律温和处理为思念、告别、关系记忆，必要时读 `safety-rules.md`。

输出提醒：不要说亲戚现实中会出事。亲戚梦更适合问“梦里的感觉是亲近、压力、尴尬还是陌生”。

## 旧人、前任、暗恋对象

传统高频含义：旧情绪、恋爱符号、未完成牵挂、现实关系投射。

判断区分：

- 反复梦见旧人：不直接等于还爱；传统/现代解梦站都常把它解释为旧情绪或当前关系状态被触动。
- 梦见复合或在一起：可能是对旧感觉、被看见感、青春阶段的回望。
- 已婚/有伴侣却梦见旧喜欢的人：重点不一定是那个人，而是当前身份和旧自由感之间的对比。
- 梦里得到旧人却失去现实成果：重点是代价感，可能在比较另一条人生路与现在稳定生活。
- 醒来想联系：旧情绪更强；醒来慌、愧疚、空落：现实压力、身份冲突或安全感更强。

输出提醒：避免问“是不是还爱”。改问“梦里的那个人更像本人，还是某段时期的自己？”

## 车、路、考试、学业、工作

传统高频含义：方向、进展、功名、身份、社会评价。

判断区分：

- 开车顺利、路清楚：偏方向感稳定。
- 刹不住、迷路、车坏：偏控制感不足、计划卡住、节奏过快。
- 考试、读研、学历：偏被评价、资格感、上升通道、过去未完成目标。
- 工作丢失、单位混乱：偏身份安全、收入压力、能力评价。
- 车、房、工作同时失去：不是单独财运梦，更像现实根基和自我价值的整体焦虑。

## 输出模板补强

传统层可以用这类句式：

- “传统上，蛇/水/老家这类意象常被放在变化、家宅、安全感这一组里看。”
- “民间解梦会把打死蛇理解为排除麻烦，但放到这个梦里，更重要的是还有一条跑掉，说明尾巴没完全收住。”
- “从传统象征看，房子、车子、工作都属于现实根基；它们一起消失，比‘旧人出现’更值得看。”

深挖时优先把问题问到现实层：

- “最近是不是有现实根基的压力，比如钱、工作、婚姻、家里责任？”
- “梦里最强的不是画面，而是哪种感觉？”
- “醒来后你更惦记梦里的人，还是更在意失去/逃不掉/回不去？”
