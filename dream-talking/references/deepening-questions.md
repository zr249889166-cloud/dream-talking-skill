# 梦境深挖问题库

使用方式：当梦境初判已经成立，但要更准地连接现实情绪时，从对应类型中挑 2 到 4 个问题。不要一次全部问。问题要像聊天，不要像问卷。

## 通用追问

1. 最近一两周现实里有没有明显压力、变化、争执、选择或疲惫？
2. 梦里最强的感觉是什么：害怕、开心、愧疚、怀念、尴尬、失落，还是空落？
3. 醒来后的余味是什么：想联系、松一口气、内疚、怅然，还是只是疑惑？
4. 梦里最让你记住的是人、地点、动作，还是最后的结局？

## 旧人/前任/暗恋对象

1. 现实里你最近有没有重新想起青春、自由、遗憾，或“如果当年另一种选择”的念头？
2. 梦里的那个人更像本人，还是更像某段时期的自己？
3. 梦里靠近对方时，你更开心、愧疚、紧张，还是觉得理所当然？
4. 现在的亲密关系里，最近有没有缺少新鲜感、被看见感或轻松感？

避免问：“你是不是还爱他/她？”

## 老家/学校/旧地点

1. 最近有没有想回到更熟悉、更安全、更不用解释自己的状态？
2. 梦里的老地方是温暖的、陌生的，还是有点压抑？
3. 那个地方让你想到的是家人、童年、某段关系，还是以前的自己？
4. 最近有没有现实身份变化，比如工作、婚姻、家庭责任、搬家或离别？

## 亲戚/家人

1. 梦里的亲戚给你的感觉是亲近、客气、尴尬、压力，还是陌生？
2. 最近家里有没有事情、联系、聚会、催促或责任感被提起来？
3. 你梦里更像是在被照顾、被评价、被拉回旧角色，还是只是路过？
4. 醒来后你是想念、烦躁、愧疚，还是觉得没什么？

## 失去房子/车子/工作/学业

1. 最近是不是特别在意稳定、收入、身份、成就或未来路线？
2. 梦里的“失去”让你害怕，还是反而轻松？
3. 你得到的东西和失去的东西之间，有没有一种“交换感”？
4. 现实里有没有一个选择让你感觉：要得到某样东西，就必须牺牲另一部分自己？

## 反复梦

1. 这个梦大概多久出现一次？每次情节一样，还是只保留同一个人/地点？
2. 最近一次梦和以前相比，有没有新的变化？
3. 每次醒来后的情绪是否相同？
4. 现实里有没有一个问题一直没真正收尾？

## 高风险或强烈负面梦

优先安抚，必要时读取 `safety-rules.md`。

1. 这个梦醒来后有没有持续影响你一天的状态？
2. 最近现实里压力是不是已经超过平时很多？
3. 有没有可以马上做的小事让自己先稳定一点，比如喝水、洗脸、找人说一句话？

如果用户表达现实自伤想法，不继续普通追问，转向安全支持。
