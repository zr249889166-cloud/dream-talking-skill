# 梦里有话 AI 解梦 Skill

一个可导入 AI Agent 的梦境解析 Skill，结合传统周公解梦文化、现代心理分析和短视频文案生成。

## 功能列表

- 梦境关键词提取
- 传统周公解梦解释
- 现代心理分析
- 梦境倾向判断
- 今日提醒
- 一句话总结
- 朋友圈 / 小红书 / 抖音文案
- 短视频口播稿生成

## 适合人群

- 想做 AI 自媒体号的人
- 想做梦境解析内容的人
- 想玩 AI 解梦的人
- 想研究 Agent Skills 的人

## 安装方式

Codex 用户可以使用 skill installer 从 GitHub repo path 安装：

```bash
$skill-installer install https://github.com/YOUR_USERNAME/dream-talking-skill/tree/main/dream-talking
```

也可以下载本项目 ZIP，然后导入 `dream-talking` 文件夹。

如果你的工具暂时不支持 Skill，可以复制 `dream-talking/SKILL.md` 的内容，作为系统提示词使用。

GitHub Skill 路径占位符：

```text
https://github.com/YOUR_USERNAME/dream-talking-skill/tree/main/dream-talking
```

## 使用示例

输入：

```text
我梦见自己开车掉进水里，但是人没事，旁边还有一条蛇。
```

输出示例：

```text
梦境关键词：
车 / 水 / 蛇 / 失控 / 脱险

传统解梦：
传统解梦里，车常和人生方向、事业推进有关；水常和变化、财运、情绪流动有关；蛇则常被理解为隐性变化、压力或机会。这个梦虽然有掉进水里的惊险感，但结局是人没事，所以更偏向“有惊无险”的梦境。

现代心理分析：
这个梦更像是在说你最近对某件事有一点失控感。车代表你对生活节奏和方向的掌控，掉进水里代表情绪或现实压力突然变大。但你最后没有出事，说明你潜意识里其实知道自己能扛过去。

一句话总结：
你不是掉进水里了，是最近脑子里的事情太满了。
```
